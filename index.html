Introducing Styles

Assignment Requirements

Content

In this assignment you will create a web page about yourself. You should cover the following content:

  • What is your Major? Explain why you selected that.
  • Describe any web development or programming experience you have.
  • What is your primary goal for this course?
  • Share one interesting thing about yourself.
  • Share a photo of yourself, or a photo that you took.

HTML minimum requirements

In this assignment we explore the fundamental text options for HTML coding and introduce inline styles.

  • 3 levels of headings
  • 3 paragraphs
  • a background color
  • 2 different fonts
  • 2 different text colors
  • 2 different text alignments
  • 2 different specific font sizes (apply to headings)

About this page

This page contains all of the minimum technical requirements, you can view the source code for the page by right clicking on the page and selecting view page source.
